      • resolve(
            taskTypeOrOpts: TaskType | ResolveOptions,
            _prompt?: string,
            defaultModel?: string,
            priority?: ModelPriority,
        ): string

        Synchronous resolve — always instant (reads memory/localStorage). A background refresh runs automatically when cache is stale. Falls back to defaultModel if cache is empty (first ever cold start).

        Supports two call forms: resolve('chat', prompt, defaultModel, 'quality') // positional (legacy) resolve({ TaskType: 'chat', Speed: 100 }) // options object

        Speed (0–100) ranks models by paramCount: 100 = fastest (smallest), 0 = most capable (largest). Ignored if priority is set explicitly.
